\# Mentoren-Plattform



\## Ziel

Web-App für Mentor mit \~8-20 Mentees.

Admin erstellt Aufgaben, Mentees haken täglich ab.



\## Stack

\- React + Tailwind CSS

\- Supabase (Auth + PostgreSQL)

\- Vercel (Hosting)



\## Aufgaben-Typen

Aufgaben haben eine Zielgröße z.B.:

\- "5 Seiten Koran" (type: pages)

\- "12 Minuten Koran" (type: minutes)

\- "7 Seiten Buch" (type: pages)

Mentee gibt täglich den erreichten Wert ein.



\## Rollen

\- admin: Aufgaben erstellen/bearbeiten, Mentees verwalten, Übersicht sehen

\- mentee: Eigene Aufgaben sehen, täglich abhaken



\## Regeln

\- Admin kann Aufgaben global erstellen

\- Admin kann Aufgaben für einzelne Mentees ausblenden

\- Daten bleiben täglich gespeichert (kein Überschreiben)
